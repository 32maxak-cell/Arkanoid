# Arkanoid 

Классическая аркадная игра «Арканоид» (разрушение кирпичей) на 7 языках программирования. Каждая версия поддерживает:

- **Платформа** – управление мышью или стрелками (влево/вправо)
- **Мяч** – отскок от стен, платформы и кирпичей
- **Кирпичи** – нескольких типов (обычные, прочные – несколько ударов)
- **Бонусы** (power-ups): расширение платформы, ускорение мяча, дополнительный мяч
- **Счёт, жизни, уровни** – прогрессивное усложнение
- **Физика** – изменение угла отскока в зависимости от места удара о платформу
- **Звуковые эффекты** (опционально)

## Реализации

| Язык | Технология | Особенности |
|------|------------|-------------|
| Python | Pygame | Полная графика, звук, бонусы |
| Go | Ebitengine | Производительный 2D |
| Rust | ggez | Высокая скорость |
| C# | Windows Forms | Классический Windows |
| JavaScript | HTML5 Canvas | Браузерная версия |
| TypeScript | Canvas + классы | Типизированная версия |
| Java | Swing | Кроссплатформенный GUI |
